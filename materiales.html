Temas del Curso

HTML5

Lenguaje de marcado utilizado para estructurar el contenido de las páginas web. Permite organizar textos, imágenes, enlaces y otros elementos.

  • Estructura básica de una página
  • Etiquetas principales
  • Enlaces e imágenes

CSS3

Lenguaje de estilos que permite mejorar la apariencia de las páginas web, aplicando colores, diseños y animaciones.

  • Colores y fondos
  • Diseño con Flexbox
  • Estilos responsivos

Diseño Web

Conjunto de técnicas para crear páginas web atractivas, funcionales y adaptables a diferentes dispositivos.

  • Diseño responsivo
  • Experiencia de usuario
  • Organización del contenido
